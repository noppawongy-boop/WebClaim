# Claim Log สำหรับ GitHub Pages

## วิธีอัปโหลด

1. สร้าง Repository ใหม่ใน GitHub
2. อัปโหลด `index.html` และ `data.json` ไว้ที่ระดับบนสุดของ Repository
3. ไปที่ **Settings → Pages**
4. ที่ **Build and deployment** เลือก **Deploy from a branch**
5. เลือก branch `main` และโฟลเดอร์ `/ (root)` แล้วกด **Save**

## การอัปเดตข้อมูล

- เพิ่ม แก้ไข หรือลบรายการได้จากหน้าเว็บ ข้อมูลจะเก็บใน Local Storage ของเบราว์เซอร์เครื่องนั้น
- กด **ส่งออก JSON** เพื่อสำรองข้อมูลล่าสุด
- หากต้องการให้ข้อมูลเริ่มต้นบน GitHub เปลี่ยนตาม ให้นำไฟล์ที่ส่งออกมาเปลี่ยนชื่อเป็น `data.json` แล้วอัปโหลดทับไฟล์เดิมใน Repository
- กด **นำเข้า JSON** เมื่อต้องการเปิดข้อมูลสำรองชุดอื่น

> GitHub Pages เป็นเว็บแบบ Static จึงไม่สามารถเขียนกลับ Repository โดยตรงโดยไม่มีระบบล็อกอิน/API เพิ่มเติม
